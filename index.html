Linux

Unreal makes packaging games for different platforms easy. Every game will be released on Linux first with Windows releases taking a back seat.

Gameplay Programming

I program all gameplay myself using C++ and Blueprints.

PC Advantages

Games will have options to make use of some of the PC's hardware advantages, like multiple monitors and superior performance thresholds.

VFX

I make all the simple visual effects for the games. Such as particle emitters and post process effects.

Moddable Games

The Unreal Engine is its self very conducive to modding and every effort will be made to expose the APIs needed for sidekick apps and mods.
